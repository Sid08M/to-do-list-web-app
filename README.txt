# To-Do List App

A simple web-based to-do list application built with HTML, CSS, and JavaScript.

## Features
- Add tasks to your list
- Delete tasks from the start or end
- View task count

## How to Use
1. Open `index.html` in your browser
2. Type a task and click "Add Task"
3. Use the delete buttons to remove tasks

## Files
- `index.html` - Main HTML structure
- `element.js` - JavaScript logic
